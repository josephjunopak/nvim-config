# Custom NeoVim Setup

Telescope:
```
'<leader>pf' - Find files
'<C-p>' - Only look at files in Git
'<leader>ps' - Grep >
```

Harpoon:
```
'<leader>a' - Add current file to Harpoon menu
'<C-e' - Toggle Harpoon menu

'<C-h>' - Go to file 1
'<C-t>' - Go to file 2
'<C-n>' - Go to file 3
'<C-s>' - Go to file 4
```

Undotree:
```
'<leader>u' - Opens UndoTree
```

Fugitve:
```
'<leader>gs' - Git Menu
```
